# Django Multi-Vendor Ecommerce

A multi-vendor ecommerce platform where multiple 
sellers can sell their products.

## Tech Stack
- Python
- Django
- SQLite

## Features
- User Authentication (Buyer/Seller/Admin)
- Product Management
- Category Management
- Cart System
- Order Management
- Admin Dashboard

## User Roles
- Buyer → Browse and purchase products
- Seller → Add and manage products
- Admin → Manage entire platform
